# Boundary Object

## 概要

Toitoi は「問い」を Boundary Object（境界対象）として扱います。

Boundary Object とは：

> 異なる文脈間で共有可能でありながら、
> 完全には同一化されない対象

を指します。

---

## Toitoi における意味

農業知識は：

- 気候
- 土壌
- 生態系
- 文化
- 実践

に強く依存しています。

そのため、ある地域の知識を
そのまま別地域へ適用することはできません。

Toitoi は：

> 「翻訳可能だが、普遍化されない問い」

を共有することで、
属地性を壊さずに知識を循環させることを目指します。

---

## 問いの翻訳

Toitoi において問いは：

- コピー
- 再利用
- 再解釈
- 他地域への翻訳

される可能性があります。

その過程自体が知識生成の一部として扱われます。

---

## 弱い連帯

Boundary Object により：

- 気候が異なる農地
- 異なる栽培方法
- 異なる文化圏

の間でも、
互いの文脈を破壊せず接続可能になることを目指します。
